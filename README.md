# Digital Marketing Campaign Analysis

## Project Overview

This project analyzes digital marketing campaign performance using SQL and Power BI to identify trends, evaluate campaign effectiveness, and generate actionable business insights.

## Tools Used

- Excel
- MySQL
- SQL
- Power BI
- GitHub

## Project Objectives

- Analyze marketing campaign performance
- Measure campaign efficiency using key marketing KPIs
- Identify high-performing acquisition channels
- Generate business recommendations to improve ROI

## Key KPIs

- Revenue
- Cost
- Impressions
- CTR
- CPC
- CPM
- Conversion Rate
- ROI
- ROAS

## Data Model

Galaxy Schema consisting of:

### Fact Tables

- fact_user_acquisition
- fact_traffic_acquisition
- fact_events
- fact_landing_page
- fact_country_performance
- fact_age
- fact_gender
- fact_interest

### Dimension Tables

- dim_channel
- dim_country
- dim_age
- dim_gender
- dim_interest
- dim_event
- dim_landing_page

## Key Insights

- Organic Search generated over 55% of total revenue.
- Campaign ROI exceeded 684%.
- ROAS reached 7.84x.
- Paid Search delivered the highest return on ad spend.
- High-performing channels generated most campaign revenue.

## Business Recommendations

- Increase investment in Organic Search.
- Scale high-ROAS paid campaigns.
- Optimize underperforming channels.
- Improve conversion funnel efficiency.
- Focus on ROI and ROAS driven decision-making.
