# Data Audit Report – Digital Marketing Campaign Analysis

## Project Overview

This project analyzes digital marketing campaign performance using Excel, MySQL, SQL, and Power BI. The objective is to evaluate campaign effectiveness, measure marketing ROI, and identify opportunities for optimization through data-driven insights.

---

## Dataset Summary

The dataset consists of multiple marketing performance reports covering:

* User Acquisition
* Traffic Acquisition
* Landing Pages
* Events
* Audience Segmentation
* Country Performance
* Age Demographics
* Gender Demographics
* Interest Categories

The data was imported into MySQL and transformed into a dimensional data warehouse for analysis.

## KPI Definitions

The following KPIs were developed for campaign performance analysis:

### Revenue

Total revenue generated by marketing activities.

### Cost

Total marketing spend across channels.

### Impressions

Total number of campaign impressions.

### CTR (Click Through Rate)

CTR = (Key Events / Impressions) × 100

### CPC (Cost Per Click)

CPC = Total Cost / Key Events

### CPM (Cost Per Mille)

CPM = (Total Cost / Impressions) × 1000

### Conversion Rate

Conversion Rate = (Key Events / Engaged Sessions) × 100

### ROI

ROI = ((Total Revenue − Total Cost) / Total Cost) × 100

### ROAS

ROAS = Total Revenue / Total Cost

## Assumptions and Limitations

* The source dataset did not contain a dedicated Clicks field.
* KPI calculations were implemented using formulas provided in the project requirements.
* Channel performance analysis was performed using aggregated marketing data.
* Results should be interpreted within the context of the available dataset.

## Key Findings

* Organic Search contributed over 55% of total revenue.
* Campaign ROI exceeded 684%.
* ROAS reached 7.84x.
* Paid Search delivered the highest return on ad spend.
* High-performing channels generated the majority of campaign revenue.
